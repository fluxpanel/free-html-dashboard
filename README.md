**Fluxa** is a premium, high-performance admin dashboard template designed with a focus on fluidity, modern aesthetics, and exceptional user experience. Built with pure HTML, Vanilla CSS, and JavaScript, it offers a lightweight yet powerful foundation for any data-driven application.

---

## ✨ Features

- 🌓 **Dynamic Theme**: Seamless switching between Light and Dark modes with a single click.
- 📊 **Multiple Dashboard Variants**:
    - **Analytics Dashboard**: Comprehensive overview of revenue and user behavior.
    - **Ecommerce Dashboard**: Specialized metrics for tracking sales, profits, and product performance.
    - **CRM Dashboard**: Focused on lead management, deal pipelines, and customer relationships.
    - **Finance Dashboard**: Monitor cash flow, monthly income/expenses, and budget allocation.
    - **Project Dashboard**: Manage projects, track task completion, and team collaboration.
- 💎 **Glassmorphism**: Modern UI elements featuring backdrop-filter effects and soft shadows.
- 📱 **Fully Responsive**: Optimized for all screen sizes, from mobile phones to ultra-wide monitors.
- 🎨 **Premium Aesthetics**: Vibrant gradients, custom typography (**Outfit**), and fluid micro-animations.
- 🛠️ **Lucide Icons**: Crisp, customizable SVG icons integrated throughout the interface.

## 🚀 Quick Start

To get started with Fluxa, simply clone the repository and open the `index.html` file in your preferred browser.

```bash
# Clone the repository
git clone https://github.com/fluxapanel/free-html-dashboard.git

# Navigate to the dashboard directory
cd free-html-dashboard

# Open index.html in your browser
open index.html
```
